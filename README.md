Project directory for 550 final project

## Contents and report generation

Navigate to the project directory e.g. cd <your_wd>/2026_02_LH_DATA550fp. 

Full report can be generated by entering "make" in the terminal. This should 
work for both mac and Windows operating systems. Note that the location of the seed 
voxel can be changed by specifiying make X=<x_value> Y=<y_value> Z=<z_value>, but 
that this will only alter figures and tables and the text of the report will be 
misaligned with the presented data.

The report final_report.html contains a description of the data and background 
information, the analysis method, 4 figures (3 brain image visualizations and a 
line plot) and some brief remarks on the results of the analysis.

## Source code

Source contains all R scripts, including the script for data download and analysis (01),
visualization code (02), functions called in the visualization code, the code for 
making the table (03) and the code for rendering the report (04). 

Data contains raw image data used in the report generation.

Output contains the generated table, generated figures, and other generated data objects
used in the report.
